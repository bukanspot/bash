# .bashrc
Tema emotikon pada terminal linux
<br>
# Instalasi
<i>hanya mengganti file .bashrc di direktori home dengan file .bashrc baru</i><br>
<i>Buka file dengan file editor dan ganti emotikon sesuai selera anda</i><br>

# command
```
$ cd "download folder"
$ mv <file> .bashrc
$ cp .bashrc ~
```
